Quick set-up guide

This is a quick-setup article to guide you on installing Canvas Tic Tac Toe on your server.

  1. 1. Unzip the contents of the file you downloaded and place them in a folder in your desktop, or another readily accessible folder.
  2. 2. Open or download your favorite FTP client such as FileZilla or Cyberduck, and connect to your server.
  3. 3. Navigate to a public html folder within your server (usually the folder containing your main index.html page) and create a new folder named tictactoe within.
  4. 4. Place the contents of the unzipped folder in your desktop from step one inside of the newly created folder on your server.
  5. 5. If you have followed the above instructions correctly, you should now be able to go to www.yourwebsite.com/tictactoe and be able to see a page much like this one.

If you still find you need help setting up the game on your server, feel free to send me an email.

Configuration

This explains how to change the X and O icons in the game.

  1. 1. Open the file named config.js that came with the game files.
  2. 2. In this file, find iconForX and set its value to the path to an image (e.g. 'icons/o.png'). Do the same for iconForO.
  4. If the images exist, and the path specified is correct, you should now see your new icons in the game instead of the default X's and O's.

If you do not see a config.js file then you don't have the latest version of the game, head here to update!

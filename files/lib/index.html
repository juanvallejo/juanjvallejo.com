Lib.js
Lib.js is an HTML5 animation library. It provides an easy-to-use API to create simple animations with a single line of code, or entire games with ease. See it in action by checking out demos here, or read its documentation here.
Simple Syntax
The library's extemely simple syntax allows you to animate any sprite image with a single line of code using HTML5 Canvas.
Complete Control
Lib.js allows users to control every aspect of an object's animation, from frame speed to frame frequency to even the order of the frames themselves.
Support
Lib.js is supported in all browsers compatible with HTML5. You can find the documentation here.

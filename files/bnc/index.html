Bounce is a jQuery plugin that provides the ability to animate html elements with a bounce effect. It uses an equation of the sine curve to give html elements a bouncing animation. This jQuery plugin is easy to use, and provides varying levels of animation, allowing the user to define an element's speed, height of bounce, and even give an element a negative bounce.

To see this plugin in action, click on the circle above. Sample code for this demo: 

$(document).ready(function() {
	$("#object").click(function() {
		$(this).bounce(3);
	});
});

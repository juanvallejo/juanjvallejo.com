Canvas Pong

Canvas Pong is the classic game of Pong re-created in JavaScript and HTML5. To get started playing, simply select a game-mode, choose a difficulty, and move the paddle with your keyboard's arrow keys.

To pause a game, press P Esc or R

Press again any of these keys to resume playing. To change different settings click on the Pong logo.

Setup Guide

This is a quick article to guide you on setting up Canvas Pong on your website.

  1. 1. Unzip the contents of the file you downloaded and place them in a folder in your desktop, or another readily accessible folder.
  2. 2. Open or download your favorite FTP client such as FileZilla or Cyberduck, and connect to your server.
  3. 3. Navigate to a public html (www) folder within your server (usually the folder containing your main index.html page) and create a new folder named pong within.
  4. 4. Place all of the contents of the unzipped folder in your desktop from step one inside of the newly created folder on your server.
  5. 5. If you have followed the above instructions correctly, you should now be able to go to http://www.yourwebsite.com/pong and be able to see a page much like this one.

If you still find you need help setting up the game on your server, feel free to send me an email.

Extra

By default, the Up Arrow moves your (player1) paddle up and Down Arrow moves it down. For player 2, the respective keys for up and down movement are W and S.

Any of these keys can be changed from the Settings Menu which can be accessed from the main (title) screen by clicking on the Pong logo.

The Esc key can be used to quick-exit menus most of the time and it as well as P and R pause a current game. Pressing any of them a second time will resume the game.

The default settings end the game after either player, or the computer reaches 5 points. This can also be changed from the Settings Menu.
